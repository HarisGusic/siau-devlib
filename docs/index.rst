===========
SIAU devlib
===========

.. rubric:: Biblioteka za čitanje uređaja iz JSON datoteka

Biblioteka *devlib* se koristi za parsiranje JSON datoteka koje sadrže opise
uređaja, kako bi se dobila C++ reprezentacija uređaja. Funkcije su izdvojene u
zasebnu biblioteku kako bi i drugi timovi mogli koristiti ove funkcije za svoje
potrebe.
